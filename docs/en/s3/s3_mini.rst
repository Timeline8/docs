Features
------------------
* based ESP32-S3-WROOM-1
* 4MB Flash
* 2MB PSRAM
* 27x IO
* 1x RGB LED (IO47)
* ADC, DAC, I2C, SPI, UART, USB OTG
* Compatible with MicroPython, Arduino and ESP-IDF
* Default firmware: MicroPython
